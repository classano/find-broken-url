== Description ==
This plugin finds all URL´s in your post and postmeta table in your database.

**Features**

* Monitors URL´s in your posts.
* Find url that responds with status 404

**Basic Usage**

Once installed the plugin starts checking your post and postmeta table for URL´s that is broken.
You can manually start it in the plugin tab.

**Translations**

* Swedish - classano

== Installation ==

To do a new installation of the plugin, please follow these steps

1. Download the find-broken-url.zip file to your computer.
1. Unzip the file
1. Upload `find-broken-url` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
